BamTools

BamTools is a project that provides both a C++ API and a command-line toolkit for reading, writing, and manipulating BAM (genome alignment) files.


Barnett DW, Garrison EK, Quinlan AR, Strömberg MP, Marth GT

Marth Lab - Biology Dept., Boston College

Citation: Barnett DW, Garrison EK, Quinlan AR, Strömberg MP, Marth GT. BamTools: a C++ API and toolkit for analyzing and managing BAM files. Bioinformatics. 2011 Jun 15;27(12):1691-2. [link]
